AI 助手可调用工具 22

💬 交互与反馈

clarify
向用户提问，获取澄清或反馈。支持多选和开放式问题。
question* choices[]

💻 终端与进程

terminal
执行 Shell 命令。Windows 通过 bash 运行。支持前台和后台模式。
command* background timeout workdir pty notify_on_complete
⏱ 默认 180s📌 最大 600s
process
管理后台进程：查看状态、获取日志、等待完成、终止。
action* session_id* timeout limit
🎬 list|poll|log|wait|kill|write|submit|close
read_terminal
读取嵌入终端窗格当前显示内容，支持翻页查看历史输出。
start_linecount
close_terminal
关闭后台进程的终端标签页，不终止进程。
process_id*

📁 文件操作

read_file
读取文本文件带行号。支持 .ipynb, .docx, .xlsx 自动提取。
path* offset limit
📄 默认 500 行📌 最大 2000 行
write_file
覆盖式写入文件，自动创建父目录，自动语法检查。
path* content*
patch
精准查找替换。replace（单文件）/ patch（V4A 多文件补丁）。
mode* path old_string new_string replace_all

🔍 搜索与查询

search_files
搜索文件内容（正则）或按名称查找（glob）。基于 ripgrep。
pattern* target path file_glob limit output_mode
session_search
搜索历史会话。支持发现（关键词）、滚动（上下文）、读取、浏览四种模式。
query limit session_id around_message_id

🧠 记忆与技能

memory
持久化记忆管理。支持批量原子操作。
target* action content old_text operations[]
🎯 target: memory | user
skill_manage
管理技能：创建、更新、删除。支持关联文件的读写。
action* name* content category
🎬 action: create|patch|edit|delete
skill_view
查看技能内容及其关联的参考文件、模板、脚本。
name* file_path
skills_list
列出所有可用技能及描述，可按分类过滤。
category

📋 任务管理

todo
管理当前会话的待办事项。
todos[] merge
📌 pending|in_progress|completed|cancelled
cronjob
管理定时任务。支持周期/单次调度，多平台交付。
action* schedule* job_id prompt skills[] script deliver no_agent enabled_toolsets[] workdir
⏰ schedule: '30m'|'every 2h'|'0 9 * * *'
delegate_task
将任务委托给独立的子代理并行执行（最多 3 个并行）。
goal context tasks[] role
👥 role: leaf | orchestrator📦 最多 3 并行

📂 项目管理

project_create
创建桌面项目并切换当前会话。
name* path*
project_list
列出所有桌面项目（无参数）。
project_switch
切换到指定项目（按名称、slug 或 id）。
project*

🎵 其他工具

execute_code
执行 Python 脚本，可在脚本中调用 Hermes 工具。适合复杂逻辑。
code*
⏱ 5 分钟📦 50KB🔧 最多 50 次调用
text_to_speech
文字转语音：将文本转为音频文件。
text* output_path

CLI 命令参考

🌍 全局标志

--version, -V
显示版本号。
-r, --resume SESSION
按 ID 或标题恢复会话。
-c, --continue [NAME]
按名称恢复或恢复最近会话。
-w, --worktree
隔离的 git worktree 模式（并行 agent）。
-s, --skills SKILL
预加载技能（逗号分隔或多次使用）。
-p, --profile NAME
使用命名配置文件。
--yolo
跳过危险命令批准（不推荐）。
--pass-session-id
在系统提示中包含会话 ID。
